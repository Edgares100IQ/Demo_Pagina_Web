# Sitio demo para marca de ropa

Archivos creados:

- `index.html` - Página principal con login al entrar.
- `styles.css` - Estilos y animaciones.
- `script.js` - Lógica de login, productos y carrito.

Cómo probar:

1. Abrir `index.html` en el navegador.
2. Credenciales demo: usuario `user`, contraseña `demo123`, o usar "Entrar como invitado".

Notas:

- Todas las funciones son cliente-side (demo). Para producción conectar un backend seguro.
- Imágenes de ejemplo provienen de picsum.photos.
